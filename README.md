# cadet_marksmanship_app
# Cadet Marksmanship Hub
A new Flutter project.
A comprehensive digital assistant for Canadian Cadet Marksmanship and Biathlon coaches. This application digitizes the CCOBCS (Cadet Corps/Squadron) marksmanship experience, from real-time range management to automated team validation and scoring.
## Getting Started
## 🎯 Key Features
### 🏁 Range Management
*   **Relay Tracker**: Manage multiple relays with real-time status updates.
*   **Lane Assignment**: Quick-assign competitors to firing points.
*   **Live Leaderboard**: Real-time individual and unit/team rankings across all relays.
### 👥 Team & Unit Registry
*   **Rule-Enforced Validation**: Automatically validates teams against official CCOBCS rules (5 members, minimum 2 Juniors).
*   **Smart Classification**: Enter Date of Birth to automatically calculate age (as of Jan 1st) and assign Junior/Senior status.
*   **Persistent Roster**: Maintain a database of your unit's competitors, ranks, and stats.
### 📏 Scoring Calculators
*   **Grouping Calculator**: Instant classification (Distinguished, Expert, etc.) with automatic CM conversion.
*   **Competition Calculator**: 10-shot series tracker with "Inner Ten" (IT) support for tie-breaking.
*   **One-Tap Save**: Push scores directly from the calculator back to the active relay lane.
### 📚 Coach's Resource Hub
*   **Digital Rulebook**: Searchable CCOBCS Marksmanship and Biathlon rules (A-CR-CCP-177/PS-001).
*   **Range Commands**: Visual reference for the Loading and Firing sequence.
*   **Biathlon Tools**: Specialized coaching module for the biathlon discipline.

*   [Flutter SDK](https://docs.flutter.dev/get-started/install)

1. Clone the repository:
   ```bash

   ```
2. Install dependencies:
   ```bash
   flutter pub get
   ```
3. Run the application:
   ```bash
   flutter run
   ```
